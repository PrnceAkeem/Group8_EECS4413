# 6ixOutside E-Store

Multi-tier e-commerce project

The application models a sneaker and streetwear store where customers can browse, filter, purchase products, and administrators can manage inventory, users, and sales analytics.

## Project Scope

The solution is designed to satisfy the official project specification in:

- [26w -- v1 EECS_Project Specs- 2023.pdf](./26w%20--%20v1%20EECS_Project%20Specs-%202023.pdf)

Core mandatory capabilities from the spec:

- Customer auth and account management (`register`, `login`, `logout`, profile + purchase history)
- Product catalog browsing with filters/search/sort and detailed product pages
- Shopping cart management (`add`, `update quantity`, `remove`, live totals)
- Checkout with payment simulation (including rejection flow)
- Admin operations for sales history, customer management, and inventory updates
- Clear architecture using MVC + DAO + Web API mediation between frontend and backend
